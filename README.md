# Kanban - Gestió de Tasques

Aplicació web de gestió de tasques tipus Kanban desenvolupada amb HTML, CSS i JavaScript.

Permet organitzar tasques en columnes (Per fer, En curs, Fet) i, progressivament, afegir funcionalitats com persistència, filtres i estadístiques.

## Estructura del projecte

index.html  
css/estils.css  
js/script.js  
img/  
README.md

## Funcionalitats

- Crear tasques amb títol, descripció, prioritat i data límit
- Editar tasques existents reutilitzant el formulari
- Eliminar tasques amb confirmació
- Canviar l’estat de les tasques (Per fer / En curs / Fet)
- Tauler Kanban amb renderització dinàmica
- Persistència de dades amb localStorage

## Guia d’ús

1. Omple el formulari i fes clic a **Afegir tasca**
2. La tasca apareix a la columna **Per fer**
3. Utilitza el selector d’estat per moure-la entre columnes
4. Clica **Editar** per modificar una tasca existent
5. Clica **Eliminar** per esborrar-la definitivament
